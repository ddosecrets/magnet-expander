## Magnet Expander

Sometimes torrent files and magnet links have a small number of trackers, and those trackers go offline. If your torrent client is established in bittorrent's distributed hash table (DHT) you *may* be able to find peers anyway, but not all peers have DHT support enabled. You can instead add new trackers to existing torrents, but this is typically tedious. This script makes the process simple.
